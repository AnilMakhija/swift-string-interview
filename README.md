# 🎯 Swift String Interview Challenges

A hands-on Swift Playground packed with common and essential string manipulation problems — perfect for interview prep and sharpening your Swift fundamentals.

---

## 🚀 What’s Inside?

This repo contains a zipped Swift Playground:

- ✅ Real-world string problems commonly asked in tech interviews  
- ✅ Clean, well-commented Swift solutions  
- ✅ Easy-to-run format with test examples  
- ✅ Focus on Swift-native techniques and readability

---

## 🧠 Topics Covered

- String reversal  
- Character frequency and uniqueness  
- Palindrome logic  
- Substring handling  
- Anagram checks  
- Word parsing and transformation

---

## 🛠 How to Use

### Clone the repo

```bash
git clone https://github.com/AnilMakhija/swift-string-interview.git
cd swift-string-interview
```

### Unzip the playground

```bash
unzip StringSculptor.playground.zip
```

### Open it in Xcode

- Launch `StringSculptor.playground`
- Run and explore the examples
- Try tweaking problems or writing your own versions!
